# SocialConnect Frontend

## AI-Powered Marketing Automation Platform

### Tech Stack
- **Framework**: Next.js 16 with App Router
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS 4 + shadcn/ui
- **Charts**: Recharts
- **State**: Zustand
- **Animations**: Framer Motion
- **Theme**: next-themes (dark/light)

### Features
- 📊 Dashboard with real-time stats and AI insights
- 📤 Upload & Create workflow (upload → analyze → enhance → generate → review)
- ✍️ Content Studio with multi-platform, multi-tone content
- 📅 Smart scheduling with AI-recommended best times
- 📈 Analytics with engagement trends, hashtag analysis, and AI recommendations
- 📢 Campaign management with progress tracking
- 🔄 Workflow automation with templates
- 🔗 15+ platform integrations
- ⚙️ Settings with AI model management, team roles, security

### Setup
1. Install dependencies: `npm install`
2. Set environment variables in `.env`
3. Run: `npm run dev`
4. Open: http://localhost:3000

### UI Sections
- Dashboard - Stats, charts, recent posts, AI insights
- Upload & Create - Image upload, AI analysis, enhancement, content generation
- Content Studio - View/edit generated content, approve/reject
- Schedule - Calendar view, approval queue, smart time suggestions
- Analytics - Engagement trends, platform comparison, top hashtags, AI recommendations
- Campaigns - Create and manage campaigns
- Workflows - Automation templates and execution history
- Accounts - Connect 15+ social platforms
- Settings - Profile, AI models, API keys, team, security, notifications, billing
